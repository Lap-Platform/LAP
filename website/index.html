How It Works

Three steps. Zero ambiguity.

⚙️

Compile

OpenAPI specs → DocLean typed contracts. Strip the prose, keep the structure. Automatic, deterministic.

📖

Consume

Agents read structured schemas instead of documentation. Type-safe, version-aware, token-efficient.

✓

Verify

Schema diffing catches breaking changes before they hit production. Know exactly what changed and when.

The Protocol

Three levels of compression

Choose the fidelity your agent needs. From full schema to ultra-lean.

L0 Full Schema

service Stripe {
  version: "2024-01-01"
  base: "https://api.stripe.com"

  type PaymentIntent {
    id: str
    amount: int
    currency: iso4217
    status: enum(
      requires_payment_method
      succeeded | canceled
    )
  }
}

Get Started

Up and running in 30 seconds

$ pip install lap-sdk

Or with npm: npm install @lap/sdk

Read the Docs → View on GitHub 
from lap import compile_spec, Agent

# Compile any OpenAPI spec
schema = compile_spec("stripe.yaml")

# Give your agent typed contracts
agent = Agent(schema, level="L1")
result = agent.call("createPaymentIntent", {
    "amount": 2000,
    "currency": "usd"
})

# 200 tokens instead of 3,000 ✓
