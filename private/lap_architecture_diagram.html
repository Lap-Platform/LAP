🧠 Why No Retraining Needed?

📖
It's Just Text
DocLean is plain structured text. Any LLM that can read English can read it. Nothing proprietary.
📉
Simpler, Not Harder
DocLean is EASIER to parse than YAML. Flat directives vs 8 levels of nesting. Less ambiguity.
🎯
Signal, Not Noise
We remove boilerplate that confuses LLMs. The information density goes UP, not down.

❌ OpenAPI YAML — What LLM Gets Today

/org/{orgId}/projects:
  post:
    summary: "List all projects"
    operationId: "listOrgProjects"
    parameters:
      - name: orgId
        in: path
        required: true
        schema:
          type: string
          format: uuid
        description: "The org ID"
    requestBody:
      content:
        application/json:
          schema:
            type: object
            properties:
              filters:
                type: object
                properties:
                  name:
                    type: string
    responses:
      200:
        description: "Success"
        content:
          application/json:
            schema:
              ...

~45 tokens for ONE endpoint

✅ DocLean — What LLM Gets With LAP

@endpoint POST /org/{orgId}/projects
@required {orgId: str(uuid)}
@optional {filters: map}
@returns(200)
